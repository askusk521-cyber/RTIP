# 迁移计划（中文版）

该文档跟踪从 Rust 到 Python + JAX 的分阶段重写计划。

## 范围规则

- 逐步迁移，每次处理一个小模块或阶段。
- 保持 Rust 源不变，除非后续阶段明确需要对比测试或夹具。
- 优先行为等价与可测试性，而不是逐行翻译。
- 默认启用 JAX x64 以尽量匹配 Rust 的 `f64` 行为。
- 所有新的随机行为使用显式 JAX PRNG keys。
- 工作流与 IO 保持普通 Python，除非数值上需要使用 JAX 原语。
