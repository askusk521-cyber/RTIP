# IC5C02384 RTIP/JAX Input Data

Source: ACS Supporting Information `ic5c02384_si_002.xyz`.

This directory is generated by `prepare_ic5c02384.py`. It converts the ACS
multi-XYZ file into single-structure XYZ files that current RTIP/JAX workflows
can read directly.

Chinese companion: [README_ZH.md](README_ZH.md).

## Contents

- `structures/`: 104 single-structure XYZ files.
- `manifest.csv`: structure index, atom count, name, electronic energy, and
  file path.
- `reaction_manifest.csv`: reactants, products, transition states, and
  SI-electronic-energy-based reaction or relative energies grouped by reaction.
- `elements_present.txt`: elements appearing in the structures.
- `reactions/*/1.xyz` and `reactions/*/2.xyz`: reactant files usable with
  `rtip-jax synthesize --inputs`.
- `reactions/*/product.xyz` and `reactions/*/ts.xyz`: product and transition
  state reference structures from the literature SI when available.

## Minimal Example

```bash
cd JAX
python -m rtip_jax.cli synthesize \
  --inputs examples/ic5c02384/reactions/1-tBu__CO2/1.xyz examples/ic5c02384/reactions/1-tBu__CO2/2.xyz \
  --output examples/ic5c02384/reactions/1-tBu__CO2/IS.xyz \
  --dist 5.0 \
  --seed 0
```

The generated `IS.xyz` is an initial separated structure for RTIP/JAX pathway or
DeePMD pathway workflows. `product.xyz` and `ts.xyz` are literature reference
structures, not JAX-generated structures.

## Validation Notes

- Energy differences in `reaction_manifest.csv` come from electronic energies
  in the SI XYZ titles (`E = ... a.u.`), not from Gibbs free energies in the
  paper tables. They will not match the paper `Delta G` values one by one.
- RTIP/JAX is not a DFT program. Reproducing a real potential-energy surface
  requires a DeePMD model that covers `B/C/H/N/O/P/S/Si`, or another reliable
  real PES provider.
- The order of `--type-map` must match the element order used when the DeePMD
  model was trained. Do not use the alphabetical order from this file unless it
  is also the model order.

Note: `CS2` follows the name used in the SI coordinate titles and local files.
